# Pokedex
Pokedex coded with python

## Self completion
The pokedex can be completed with your own pokemon informations 
* Name
* Type 
* Category
* Movepool 
* Evolution 
* Pokedex entry
* Image (Add an image to "images" folder then: ./images/namefile.png)

## Available pokemon 
* Impidimp
* Morgrem
* Grimmsnarl
* Budew
* Roselia
* Roserade
